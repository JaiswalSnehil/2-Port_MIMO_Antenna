# 2-Port_MIMO_Antenna
Design of 2-Port Microstrip Patch MIMO Antenna for Sub-6 GHz 5G Applications

This repository presents a	compact two port Sub-6GHz MIMO antenna developed for emerging 5G wireless communication links. The proposed structure is fabricated on an FR-4 substrate (εr = 4.4,	thickness = 1.6 mm) with an overall size of 60 × 33 mm². The antenna operates from 3.9–6 GHz, covering major 5G bands including n77, n78, and n79. A modified crescent-shaped radiator is employed to enhance impedance bandwidth and radiation performance, while structural isolation techniques are incorporated to reduce mutual coupling between ports. Simulated results show good impedance matching with return loss below −10 dB, isolation greater than 21 dB, and VSWR below 2 (≈ 1.12) throughout the operating band. The antenna provides a peak realized gain of 3.9 dBi with stable radiation behavior. The minimal ECC value ( ≈ 0.0001) validates that the antenna elements operate with high degrees of independence. Given its miniaturized geometry and high isolation levels, the presented MIMO configuration is well-suited for next-generation 5G mobile communications.
